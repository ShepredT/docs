    笔记

    Java 8 之后就不在有 JRE 了.如果想要在高级版本中使用JRE需要通过 Doc 命令进行解压

    • 第一步

    使用 Windows+R 调出 输入 CMD

    • 第二步

    用你的 CMD 找到你安装 JDK 的目录 输入下方指令即可将 JRE 解压出来 

    bin\jlink.exe --module-path jmods --add-modules java.desktop --output jre
