    多线程对话

    玩家可以通过 BetonQuests 多线程对话系统 与 NPC 进行对话.玩家可以自定义多线程的对话内容，用Npc对话叙述故事，并影响故事进度，提供多个结局或多种选择性。视频中的对话展示内容只是其中的一种方式，还有其他样式提供选择。
