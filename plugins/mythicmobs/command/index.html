    指令篇（Mythicmobs）

    MythicMobs 有简单的指令结构，你只需要输入 /mythicmobs（/mm） 。输入以后会显示一个表格，列出了所有可用的命令，每个美玲也会提供更加详细的信息来指导你如何使用。参数说明：[]为必需，<>可选

    基础命令

    mythicmobs

    /mm - 插件基础命令，显示所有的命令

    debug

    /mm debug [等级] - 设置插件的Debug日志等级。 0 = OFF

    debugmode

    /mm debugmode [true/false] - 启用Debug模式，这将停用随机生成、生物刷怪笼、和其他随机的能让Debug变难的东西

    reload

    /mm reload - 重载插件

    save

    /mm save - 强制存储所有活跃的生物

    物品命令

    完整命令格式: /mythicmob items [对应的指令]

    简写命令格式: /mm i [对应的指令]

    例: /mm i list

    items

    /mm items

    所有有关物品的命令

    get

    /mm items get [物品名] <数量>

    给你一个在配置中的怪物装备物

    give

    /mm items give [玩家] [物品名] <数量>

    给指定玩家一个在配置中的怪物装备物

    list

    /mm items list

    列出所有配置的物品
