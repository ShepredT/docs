    如何创建一个副本地图

    用到的指令如下:

    /dxl create [地图名称] - 创建一个副本地图
/dxl leave - 离开当前副本地图
/dxl import [地图名称] - 从外部导入地图成为副本地图
/dxl edit [地图名称] - 编辑一个现有的副本地图

    设置游戏的细节

    通过告示牌 如图下解释的内容

    • 第一行内容解释

      格式: [Type] - 指代第一行是什么类型的模式 如图例: [Mob] 生成生物

    • 第二行内容解释

      与你设置的第一行的内容相关的特定属性设定 如图列: Zombie 会生成 僵尸

      这里有两个参数 1 和 10 分别代表着 <数量> 和 <延迟>（间隔多久才会刷出僵尸）

    • 第四行内容解释

      触发器 触发的方式有很多中 这里 D 半径 (当玩家靠近多少格的时候出发刷出僵尸)
